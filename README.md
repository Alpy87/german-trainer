# German Trainer

An interactive, AI-powered German language learning web application designed for A1 (Beginner) students. Built with Next.js 15, React, and the Vercel AI SDK.

## Features
- 🗣️ **Conversation Partner**: Practice basic conversations with an AI that gently corrects your mistakes.
- 📖 **Grammar Coach**: Learn basic grammar rules (pronouns, cases, verb conjugation) with targeted exercises.
- 🎧 **Vocabulary Builder**: Learn new words with spaced repetition and mini-stories for stubborn words.
- 🌍 **Scenario Simulator**: Practice real-life situations (e.g., ordering at a café).
- ✍️ **Writing Tutor**: Practice writing short messages and receive a 4-axis score with a polished rewrite.

## Getting Started

1. Set up your environment variables:
   Copy `.env.local.example` to `.env.local` and add your API keys.

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
